## ✨ Features

- **Window Management**: Fully draggable, resizable, minimizable, and focusable windows powered by `react-rnd`
- **Dynamic Dock**: Magnification effect, bounce animations, and active app indicators
- **Top Menu Bar**: Apple-style menu with app-specific options, real-time clock, and system status
- **Built-in Apps**:
  - **Terminal** — Interactive CLI simulation (`react-console-emulator`)
  - **Notes** — Markdown editor with live preview (`react-markdown`)
  - **Calculator** — Pixel-perfect macOS-style calculator
  - **GitHub** — Repository viewer and stats integration
  - **Resume / Portfolio** — Dedicated professional profile window
- **Dynamic Wallpapers**: Multiple rotating backgrounds with persistence
- **Context Menu**: Right-click actions on desktop and windows
- **Spotlight-like Search** (optional extension)
- **Control Center** elements (volume, brightness toggles)

## 🚀 Getting Started

### Prerequisites
- Node.js 18+
- npm / yarn / pnpm

### Installation

```bash
git clone https://github.com/yourusername/macfolio.git
cd macfolio
npm install

### Development

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) to view it in the browser.

### Build for Production

```bash
npm run build
```

The output will be in the `dist/` folder.
